# Learn JavaScript: The Complete Developer Guide 🚀

Welcome to the **Learn JavaScript** repository! This guide is designed to take you from a complete beginner to an advanced JavaScript developer, with a special emphasis on **Functional Programming (FP)**. 

Whether you are just starting out or looking to refine your problem-solving skills, this repository provides a structured, step-by-step curriculum with real-world analogies, mental models, and hands-on coding challenges.

---

## 🎯 What You Will Learn
- **Core JavaScript Foundations**: Variables, scopes, types, closures, and the event loop.
- **Problem Solving**: 100 progressively difficult coding challenges to test your skills.
- **Functional Programming (FP)**: Pure functions, immutability, currying, and monads.
- **Advanced Patterns**: How to apply FP to real-world problem solving.

---

## 📖 The Learning Path

To get the most out of this repository, we recommend following the chapters in numerical order:

### [Chapter 1: JavaScript Foundations](./01-javascript-foundations.md)
Start here! A comprehensive reference for all core JavaScript concepts. Includes memory aids and analogies (like the "Backpack" analogy for closures and the "Elevator" for hoisting) to help you build strong mental models.

### [Chapter 2: JavaScript Coding Problems](./02-javascript-coding-problems.md)
Practice what you learned in Chapter 1 with 100 coding challenges ranging from absolute beginner to advanced system design.

### [Chapter 3: Functional Programming Guide](./03-functional-programming-guide.md)
Dive into the paradigm of Functional Programming in JavaScript. Learn how to write predictable, bug-free code using pure functions, immutability, and higher-order functions.

### [Chapter 4: Functional Programming Problems](./04-functional-programming-problems.md)
Put your new FP skills to the test with specialized challenges focusing on pure functions, `map`/`filter`/`reduce`, currying, and recursion.

### [Chapter 5: Problem Solving with FP](./05-problem-solving-with-fp.md)
Advanced problem-solving strategies using Functional Programming. Learn how to compose complex pipelines and use functional patterns in everyday code.

### [Chapter 6: Quick Reference Card](./06-quick-reference-card.md)
Your go-to cheat sheet. Use this to quickly recall syntax, mnemonics, and key takeaways when you're stuck or preparing for an interview.

---

## 💡 How to Use This Guide

1. **Read the Guides**: Start with the foundational chapters (01 and 03). Pay special attention to the highlighted `> **Analogies**` and `> **🔀 Functional Perspectives**`.
2. **Solve the Problems**: When tackling chapters 02 and 04, try solving the problems in your own editor before looking at the hints or solutions.
3. **Review the Cheat Sheet**: Keep Chapter 06 handy as a quick reference while you code.

---

## 🛠 Prerequisites

- Basic understanding of what programming is.
- A code editor (like [VS Code](https://code.visualstudio.com/)).
- Node.js installed to run the code snippets locally, or simply use your browser's Developer Console.
